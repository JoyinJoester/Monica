# 🗺️ Monica 全体ドキュメントマップ

::: note 概要
`Monica` ローカルパスワードマネージャーの公式ドキュメントセンターへようこそ。このページでは、パスワードマネージャーのエコシステム全体のシステムアーキテクチャ、コアガイド、および開発仕様への迅速なアクセスルートを提供します。

本セクションでは、プロジェクトの基本紹介、全体的な設計理念、および一般ユーザー向けの日常的な利用手順を案内します。

* **[🎯 機能総覧](/ja/guide/overview)** * *クライアントおよびブラウザ拡張機能のコア機能の境界、および UI レイアウトを迅速に把握できます。*
