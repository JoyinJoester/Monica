# Monica 自动填充设备特定设置指南

## 概述

不同的 Android 设备制造商对系统进行了定制，可能需要额外的配置才能让自动填充功能正常工作。本指南提供针对各主流品牌的详细设置步骤。

---

## 小米 (MIUI / HyperOS)

### 系统版本
- MIUI 12+
- HyperOS 1.0+

### 启用自动填充

#### 步骤 1: 系统设置
1. 打开 **设置**
2. 进入 **更多设置** → **语言和输入法**
3. 点击 **自动填充服务**
4. 选择 **Monica**

#### 步骤 2: 安全中心权限
1. 打开 **安全中心**
2. 进入 **应用管理** → **权限管理**
3. 找到 **Monica**
4. 允许以下权限：
   - ✅ 自启动
   - ✅ 后台弹出界面
   - ✅ 显示悬浮窗

#### 步骤 3: 电池优化
1. 打开 **设置** → **应用设置** → **应用管理**
2. 找到 **Monica**
3. 点击 **省电策略**
4. 选择 **无限制**

### 常见问题

**Q: 自动填充偶尔不工作？**
- 确保 Monica 在后台运行
- 检查是否被 MIUI 的内存清理杀死
- 在最近任务中锁定 Monica

**Q: 内联建议不显示？**
- MIUI 12 需要 Android 12+
- HyperOS 支持更好
- 如果不支持会自动降级到下拉菜单

### 推荐设置
- 超时时间：3000ms（已自动优化）
- 重试次数：2 次（已自动优化）

## 华为 (EMUI / HarmonyOS)

### 系统版本
- EMUI 10+
- HarmonyOS 2.0+

### 启用自动填充

#### 步骤 1: 系统设置
1. 打开 **设置**
2. 进入 **系统和更新** → **语言和输入法**
3. 点击 **自动填充服务**
4. 选择 **Monica**

#### 步骤 2: 手机管家权限
1. 打开 **手机管家**
2. 进入 **应用启动管理**
3. 找到 **Monica**，点击右侧开关
4. 选择 **手动管理**
5. 允许：
   - ✅ 自启动
   - ✅ 关联启动
   - ✅ 后台活动

#### 步骤 3: 电池优化
1. 打开 **设置** → **电池** → **应用启动管理**
2. 找到 **Monica**
3. 关闭 **自动管理**
4. 手动允许所有权限

### 常见问题

**Q: HarmonyOS 不支持内联建议？**
- 是的，HarmonyOS 目前不完全支持
- Monica 会自动使用下拉菜单模式
- 功能完全正常，只是显示方式不同

**Q: 自动填充经常失效？**
- 华为的后台限制较严格
- 确保完成所有权限设置
- 定期检查 Monica 是否在后台运行

### 推荐设置
- 超时时间：4000ms（已自动优化）
- 不支持内联建议（自动降级）

## 通用建议

### 所有设备都应该做的设置

1. **启用自动填充服务**
   - 在系统设置中选择 Monica

2. **允许后台运行**
   - 防止系统杀死 Monica 进程

3. **关闭电池优化**
   - 确保 Monica 能及时响应

4. **锁定最近任务**
   - 防止被清理

### 验证设置是否正确

1. 打开 Monica 应用
2. 进入 **设置** → **自动填充**
3. 查看状态指示器：
   - ✅ 绿色 = 一切正常
   - ❌ 红色 = 需要配置

4. 点击 **故障排查** 查看详细信息

### 测试自动填充

1. 打开任意应用（如微信）
2. 点击登录表单
3. 应该看到 Monica 的密码建议
4. 如果没有，查看 [故障排查指南](troubleshooting-guide.md)
