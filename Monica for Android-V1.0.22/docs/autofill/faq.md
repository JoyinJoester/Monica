# Monica 自动填充常见问题 (FAQ)

## 基础问题

### Q1: 什么是自动填充？

自动填充是 Android 系统提供的功能，允许密码管理器在您填写表单时自动提供用户名和密码建议，无需手动复制粘贴。

### Q2: Monica 的自动填充支持哪些 Android 版本？

- **最低要求**: Android 8.0 (API 26)
- **推荐版本**: Android 11+ (支持内联建议)
- **最佳体验**: Android 12+

### Q3: 自动填充安全吗？

是的，非常安全：
- ✅ 所有密码都经过加密存储
- ✅ 需要生物识别或主密码验证
- ✅ 密码不会发送到服务器
- ✅ 只在您授权时填充
- ✅ 符合 Android 安全标准

### Q4: 自动填充会消耗更多电量吗？

不会。Monica 的自动填充服务：
- 只在需要时激活
- 不会常驻后台
- 经过性能优化
- 电量消耗可忽略不计

## 设置和配置

### Q5: 如何启用自动填充？

1. 打开 Monica 应用
2. 进入 **设置** → **自动填充**
3. 点击 **前往系统设置**
4. 选择 **Monica** 作为自动填充服务

详细步骤请参考 [快速入门指南](quick-start-guide.md)。

### Q6: 可以同时使用多个自动填充服务吗？

不可以。Android 系统一次只允许一个自动填充服务。如果您想使用 Monica，需要禁用其他自动填充服务（如 Google 自动填充）。

### Q7: 如何为密码设置应用包名？

1. 编辑密码条目
2. 找到 **应用包名** 字段
3. 点击选择器图标
4. 从列表中选择应用
5. 或手动输入包名（如 `com.tencent.mm`）

### Q8: 域名匹配策略有什么区别？

- **精确匹配**: 只匹配完全相同的域名
  - 例如：`github.com` 只匹配 `github.com`
  
- **子域名匹配** (推荐): 匹配子域名
  - 例如：`github.com` 匹配 `github.com` 和 `gist.github.com`
  
- **基础域名匹配**: 匹配基础域名
  - 例如：`github.com` 匹配所有 `.github.com` 域名

## 使用问题

### Q9: 为什么有些应用看不到密码建议？

可能的原因：
1. **没有匹配的密码** - 添加对应的密码条目
2. **字段识别失败** - 应用使用了自定义表单
3. **应用不兼容** - 某些应用禁用了自动填充
4. **包名不匹配** - 检查密码条目的应用包名

### Q10: 为什么浏览器中看不到密码建议？

确保：
1. 密码条目填写了 **网站** 字段
2. 网站地址正确（如 `github.com`）
3. 不要包含 `https://` 或 `www.`
4. 域名匹配策略设置合适

### Q11: 什么是内联建议？

内联建议是 Android 11+ 的新功能，密码建议直接显示在键盘上方，无需切换界面。

**要求：**
- Android 11+
- 兼容的设备和 ROM
- 支持的输入法

**不支持时：**
Monica 会自动降级到传统的下拉菜单模式，功能完全正常。

### Q12: 如何保存新密码？

**自动保存：**
1. 在应用或网站中输入新密码
2. 点击登录
3. Monica 会弹出保存提示
4. 点击 **保存**

**手动添加：**
1. 在 Monica 中点击 **+** 按钮
2. 选择 **密码**
3. 填写信息并保存

### Q13: 可以在自动填充时编辑密码吗？

不可以。自动填充时只能选择现有密码。如需编辑：
1. 打开 Monica 应用
2. 找到对应的密码条目
3. 编辑并保存
4. 下次自动填充时会使用新密码

## 兼容性问题

### Q14: 哪些设备品牌支持最好？

**完美支持：**
- Google Pixel (原生 Android)
- 三星 (One UI)
- 一加 (OxygenOS)

**良好支持：**
- 小米 (MIUI/HyperOS)
- OPPO (ColorOS)
- Realme (Realme UI)
- Vivo (OriginOS)

**需要额外配置：**
- 华为 (HarmonyOS/EMUI)

详见 [设备特定指南](device-specific-guides.md)。

### Q15: 为什么华为设备不支持内联建议？

HarmonyOS 对 Android 自动填充框架进行了定制，目前不完全支持内联建议。Monica 会自动使用下拉菜单模式，功能完全正常。

### Q16: 小米设备为什么需要额外权限？

MIUI 的安全机制较严格，需要：
- 允许自启动
- 允许后台弹出界面
- 关闭省电优化

这些设置确保 Monica 能及时响应自动填充请求。

### Q17: 哪些应用不支持自动填充？

某些应用可能不支持：
- 使用自定义键盘的应用
- 明确禁用自动填充的应用
- 使用 WebView 但未正确配置的应用
- 某些游戏和特殊应用

## 性能和优化

### Q18: 自动填充响应很慢怎么办？

**优化建议：**
1. 减少密码条目数量（建议 < 500）
2. 清理设备存储空间
3. 关闭不必要的后台应用
4. 检查设备特定设置

详见 [故障排查指南](troubleshooting-guide.md)。

### Q19: 为什么有时需要等待几秒？

不同设备的响应时间不同：
- **原生 Android**: 通常 < 1 秒
- **国产 ROM**: 可能需要 2-4 秒
- **低端设备**: 可能更长

Monica 已针对不同设备优化了超时时间。

### Q20: 如何提高匹配准确度？

1. **正确填写信息**
   - 应用：填写应用包名
   - 网站：填写域名

2. **使用合适的匹配策略**
   - 推荐：子域名匹配

3. **整理密码条目**
   - 删除重复的密码
   - 使用清晰的标题

## 安全和隐私

### Q21: 自动填充会泄露密码吗？

不会。Monica 的安全措施：
- 密码加密存储
- 需要验证才能填充
- 不会自动填充到未知应用
- 不会发送数据到服务器

### Q22: 生物识别验证安全吗？

是的。生物识别数据：
- 存储在设备的安全区域
- 不会被 Monica 访问
- 不会被上传
- 由 Android 系统管理

### Q23: 可以禁用某些应用的自动填充吗？

目前不支持。但您可以：
- 不为该应用添加密码
- 或在自动填充时选择 "不填充"

### Q24: 自动填充会记录我的输入吗？

不会。Monica 只会：
- 检测表单字段类型
- 提供密码建议
- 不会记录您的输入内容

1. 添加多个密码条目
2. 使用不同的标题区分
3. 自动填充时会显示所有匹配的密码
4. 选择您想使用的账号

### Q32: 可以自定义自动填充行为吗？

目前支持的自定义选项：
- 域名匹配策略
