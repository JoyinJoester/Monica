## 📑 项目信息

::: info 核心要素
| 属性 | 内容 |
| :--- | :--- |
| **项目类型** | 本地优先开源密码管理器 |
| **核心格式** | MDBX（加密密码数据库） |
| **支持平台** | Android 8.0+ / Web |
| **开源协议** | GNU GPL v3.0 |
| **设计目标** | 离线可用、长期存档、多设备同步 |
| **安全基线** | Argon2id + XChaCha20-Poly1305 + Tiga 三档模式 |
:::
