# Monica 自动填充快速入门指南

## 简介

Monica 密码管理器提供强大的自动填充功能，可以在您使用应用和浏览网页时自动填充用户名和密码，让您的登录体验更加便捷和安全。

## 功能特性

- ✅ 自动识别登录表单
- ✅ 支持原生应用和浏览器
- ✅ 支持 15+ 种语言的字段识别
- ✅ 智能密码匹配
- ✅ 内联建议（Android 11+）
- ✅ 支持多种字段类型（用户名、邮箱、电话、地址等）

## 启用自动填充服务

### 第一步：在系统设置中启用

1. 打开 Monica 应用
2. 进入 **设置** → **自动填充**
3. 点击 **前往系统设置** 按钮
4. 在系统设置中选择 **Monica** 作为自动填充服务

### 第二步：验证服务状态

返回 Monica 应用，您应该看到：
- ✅ 绿色的状态指示器
- ✅ "自动填充已启用" 的提示

如果看到红色警告，请参考 [故障排查指南](troubleshooting-guide.md)。

## 使用自动填充

### 在原生应用中使用

1. 打开任意应用（如微信、支付宝等）
2. 点击登录表单的用户名或密码字段
3. 您会看到 Monica 的密码建议
4. 点击您想使用的密码
5. 完成生物识别验证（如果启用）
6. 密码将自动填充到表单中

### 在浏览器中使用

1. 打开浏览器（Chrome、Firefox 等）
2. 访问任意需要登录的网站
3. 点击登录表单
4. 选择 Monica 提供的密码建议
5. 自动完成登录

### 内联建议（Android 11+）

在支持的设备上，密码建议会直接显示在键盘上方，无需切换界面。

## 添加新密码

### 方法一：手动添加

1. 在 Monica 应用中点击 **+** 按钮
2. 选择 **密码**
3. 填写以下信息：
   - **标题**：密码的名称（如"微信"）
   - **用户名**：您的账号
   - **密码**：您的密码
   - **网站**：网站地址（用于浏览器匹配）
   - **应用包名**：应用的包名（用于原生应用匹配）
4. 点击 **保存**

### 方法二：自动保存

当您在应用或网站中输入新密码时，Monica 会提示您保存：
1. 输入用户名和密码
2. 点击登录按钮
3. Monica 会弹出保存提示
4. 点击 **保存** 即可

## 配置选项

### 自动填充设置

在 **设置** → **自动填充** 中，您可以配置：

- **启用自动填充**：开启或关闭自动填充功能
- **域名匹配策略**：
  - **精确匹配**：只匹配完全相同的域名
  - **子域名匹配**：匹配子域名（推荐）
  - **基础域名匹配**：匹配基础域名
- **请求保存数据**：在输入新密码时提示保存
- **生物识别验证**：使用指纹或面部识别保护密码

### 应用关联

为了让自动填充更准确，建议为每个密码设置应用包名：

1. 编辑密码条目
2. 点击 **应用包名** 字段
3. 从列表中选择应用，或手动输入包名
4. 保存

**常见应用包名：**
- 微信：`com.tencent.mm`
- 支付宝：`com.eg.android.AlipayGphone`
- 淘宝：`com.taobao.taobao`
- 京东：`com.jingdong.app.mall`

## 常见问题

### Q: 为什么没有看到密码建议？

**可能的原因：**
1. 自动填充服务未启用 → 检查系统设置
2. 没有匹配的密码 → 添加对应的密码条目
3. 字段识别失败 → 尝试手动点击字段
4. 应用不兼容 → 查看兼容性列表

### Q: 如何为浏览器中的网站添加密码？

在添加密码时，填写 **网站** 字段：
- 输入完整的域名，如 `github.com`
- 不需要包含 `https://` 或 `www.`
- Monica 会自动匹配该网站

### Q: 内联建议不显示怎么办？

内联建议需要：
- Android 11 或更高版本
- 兼容的输入法
- 部分国产 ROM 可能不支持

如果不支持，Monica 会自动降级到下拉菜单模式。

### Q: 如何在多个设备上同步密码？

使用 Monica 的云同步功能：
1. 进入 **设置** → **备份与同步**
2. 配置 WebDAV 或其他同步方式
3. 在其他设备上登录相同账号
